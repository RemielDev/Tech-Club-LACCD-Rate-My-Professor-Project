Welcome! This extension automatically shows Rate My Professor ratings
for instructors on LACCD course search pages.

────────────────────────────────────────────────────────────────────────
  QUICK START (5 Minutes)
────────────────────────────────────────────────────────────────────────

1. GENERATE ICONS (if not already done)
   - Icons already created in icons/ folder!
   - Skip this step if you see 3 PNG files in icons/

2. LOAD EXTENSION
   - Open Chrome
   - Go to: chrome://extensions/
   - Enable "Developer mode" (top-right toggle)
   - Click "Load unpacked"
   - Select THIS folder
   - Extension should appear with no errors

3. TEST IT
   - Visit: https://mycollege-guest.laccd.edu/psc/classsearchguest/EMPLOYEE/HRMS/c/COMMUNITY_ACCESS.CLASS_SEARCH.GBL
   - Search for any classes (try "ENGLISH")
   - Wait 5 seconds after results load
   - See ⭐ ratings appear next to instructor names!

────────────────────────────────────────────────────────────────────────
  TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────

Nothing showing?
  → Wait 10 seconds
  → Refresh page (F5)
  → Check console (F12) for errors
  → Read QUICK_START.md → Troubleshooting

Extension won't load?
  → Check Developer mode is ON
  → Verify icons exist in icons/ folder
  → Look for errors on chrome://extensions/
  → Read INSTALLATION.md

Some professors show, some don't?
  → That's normal!
  → Some professors aren't on Rate My Professor yet
  → "Not Found" means they're new or not rated
